# 计算机视觉概论 (Introduction to Computer Vision)

## 1. 相关学科：计算机图形学 vs. 计算机视觉

- **计算机图形学 (Computer Graphics)**：这是一个从 **模型 (model)** 到 **图像 (image)** 的过程。它的任务是利用数学模型和算法来创造、生成视觉内容。
- **计算机视觉 (Computer Vision)**：这是一个从 **图像 (image)** 到 **模型 (model)** 的过程。它的核心目标是让计算机能够像人类一样"看懂"和理解图像与视频的内容。

## 2. 什么是数字图像？

- **定义**：一张数字图像可以看作是一个二维的像素（Pixel）网格阵列。
- **什么是像素 (Pixel)**：一个像素是在图像某个特定位置上对连续颜色函数的一次采样。
- **像素值**：像素的数值（如灰度值）代表了其亮度或颜色。通常，数值越大，像素越亮；数值越小，像素越暗。

## 3. 自然图像的基本属性

### 相邻像素是相似的 (Nearby pixels are similar)

- **原因**：这源于物理世界和成像原理。真实世界中的物体表面大多是连续的。通过针孔相机模型成像时，物体上相邻的点会投射到成像平面（图像）上相邻的像素位置。
- **直观表现**：在自然图像中，像素值的变化通常是平滑和渐变的。而在随机生成的噪点图中，相邻像素则毫无关联。
- **应用**：这个属性是许多图像处理和视觉算法的基础。例如，我们可以通过检测颜色或亮度**突变 (abrupt changes)** 的地方来寻找物体的边缘和轮廓。
- **反例**：伪装 (Camouflage) 是一个特例，动物的肤色和环境高度相似，导致物体边缘的颜色变化不明显，从而迷惑观察者。

## 4. 计算机视觉的主要研究任务

随着深度学习技术的发展，许多经典视觉任务的效果得到了极大的提升。主要任务包括：

1. **图像分类 (Image Classification)**：判断一张图像属于哪个类别（如猫、狗、杯子）。
2. **目标检测 (Object Detection)**：在图像中定位出不同物体的位置（通常用边界框），并识别它们的类别。
3. **图像分割 (Image Segmentation)**：在像素级别对图像进行划分，将属于同一物体的像素区域分割出来。
4. **图像描述 (Image Captioning)**：用一句自然语言来描述图像中的内容。
5. **图像生成 (Image Generator)**：根据输入（如标签、草图、文字描述）生成全新的、逼真的图像。
6. **视频分析 (Video Analysis)**：对视频序列进行分析，如动作识别、目标跟踪等。

## 5. 图像分类：核心任务与挑战 (Image Classification: The Core Task and Its Challenges)

### 5.1 任务定义

- **图像分类**是计算机视觉中最基础、最核心的任务之一。

- **目标**：给定一张输入的图像，从一个预定义的类别列表中，为该图像分配一个最合适的标签。

### 5.2 主要挑战

计算机"看懂"图像之所以困难，是因为同一物体在图像中可以呈现出千变万化的形式：

- **视点变化 (Viewpoint Variation)**：物体可以从不同角度拍摄。

- **尺度变化 (Scale Variation)**：物体在图像中的尺寸大小不一。

- **类内变化 (Intra-class Variation)**：同一类别内的物体外观差异很大（例如，不同品种、形态的椅子）。

- **图像变形 (Deformation)**：物体可以呈现出各种姿态和形变。

- **图像遮挡 (Occlusion)**：物体可能被部分遮挡。

- **照明条件 (Illumination Variation)**：光照的强度、方向、颜色都会极大地影响图像的像素值。

- **背景杂斑 (Background Clutter)**：物体可能与背景融为一体，难以区分。

### 5.3 机器学习范式

- **监督学习**：最主流的方法。模型从大量**带有标签**的图像中学习。

- **非监督学习**：模型从**无标签**的数据中学习内在的结构或模式。

- **半监督学习**：利用少量有标签数据和大量无标签数据进行学习，在标注成本高昂时非常有用。

### 5.4 关键数据集

图像分类的发展离不开大规模基准数据集的推动：

- **MNIST**：手写数字识别的入门级数据集。

- **CIFAR-10/100**：包含了10或100个类别的低分辨率彩色图像，是研究和实验中常用的数据集。

- **ImageNet**：拥有超过1400万张图像，涵盖超过2万个类别。其子集（1000类，120万张图）上的**大规模视觉识别挑战赛 (ILSVRC)** 直接推动了AlexNet的诞生和整个深度学习的复兴。目前，计算机在该任务上的分类水准已超过人类。

## 6. 图像分类方法演进：从手动特征到端到端学习 (Evolution of Image Classification: From Manual Features to End-to-End Learning)

### 6.1 传统方法：基于特征的学习

在深度学习流行之前，传统的图像分类是一个多阶段的过程：

1. **特征提取 (Feature Extraction)**：

   - 这是最关键的一步，需要领域专家**手动设计算法**来提取有区分度的特征。

   - 这些算法基于先验知识，用于度量图像的特定内容，如边缘、角点、纹理等。

   - **常用特征**：SIFT (尺度不变特征变换), HOG (方向梯度直方图), LBP (局部二值模式) 等。

2. **分类模型训练**：

   - 将提取出的特征向量输入到一个经典的机器学习分类器中进行训练。

   - **常用分类器**：支持向量机 (SVMs), 随机森林 (Random Forests) 等。

**缺点**：特征设计的好坏直接决定了系统性能的上限，且过程繁琐，泛化能力有限。

### 6.2 深度学习方法：端到端学习

采用深度学习（特别是CNN）的方法，彻底改变了这一流程：

- **端到端模型 (End-to-End Model)**：

  - 模型直接将**原始的像素数据**作为输入。

  - 跳过了手动设计特征的步骤。

  - 网络通过其深层的结构，**自动地从数据中学习**有用的特征。隐藏层中的卷积核（kernel/filter）就是学习到的特征检测器。

  - 特征的提取和分类器的训练被整合在一个统一的框架内，通过优化同一个损失函数来共同完成。

- **输出**：网络的最终输出是每个类别标签上的概率分布。

**优势**：CNN能够学习到比手动设计的特征更强大、更抽象、更具区分度的特征表示，从而在各项视觉任务上取得了突破性的性能。
